ResponSlider.js showcase

Here are some examples where you can see how this brand-new responsive slider works.

Example 1 Hello-world slider

Let's build a simple 400px-tall slider with 3 visible slides. Every time we click on a Previous slide or Next slide button, it will just move 1 slide.

Here's the code for the <head> section

<style type="text/css">                              
  .mySlider {
    height: 400px;
    margin: 20px;
  }

  .responsive-slide {
    padding: 20px;
  }
</style>

We should place our slider's <div> anywhere within the <body>

<div id="slider" class="mySlider">

  <img src="...">

  <img src="...">

  ...

</div>

We include our scripts near the bottom of the <body>

<script type="text/javascript" 
  src="https://ajax.googleapis.com/ajax/libs/jquery/1/jquery.min.js"></script>
<script type="text/javascript" src="responSlider.js"></script>

<script type="text/javascript">
  $("#slider").responSlider({
      movingSlides: 1,
      visibleSlides: 3,
      previousSlideAction: "#previousSlide",    
      nextSlideAction: "#nextSlide"   
  });
</script>

And here's the actual slider

Resize your browser window so you can see how the slider adapts itself to its container's width.

Example 2 Multi-movement, centered slider

Now, we'll create a slider with 2 visible slides, that moves 2 slides on every click. These slides should be horizontally and vertically centered.

We just have to change the JavaScript code a little bit:

$("#slider2").responSlider({
    movingSlides: 2,
    visibleSlides: 2,
    previousSlideAction: "#previousSlide2",    
    nextSlideAction: "#nextSlide2",
    horizontallyCentered: true,
    verticallyCentered: true   
});
Previous slide Next slide

If you resize your browser window, you can see that images always stay in the center, even if they are taller than the slider

Example 3 Media-query-dependant slider

What about a media-query-dependant slider? Let's say we want a 3-slide slider if the viewport is over 768px width, but if it's smaller, it should transform into a 1-slide slider.

$("#slider3").responSlider({
    movingSlides: 3,
    previousSlideAction: "#previousSlide3",    
    nextSlideAction: "#nextSlide3",
    horizontallyCentered: true,
    verticallyCentered: true,
    mediaQueries: {
      "@media screen and (min-width: 768px)": 3,
      "@media screen and (max-width: 768px)": 1,
    }
});
Previous slide Next slide

Resize your browser window and see how the slider transforms!

Example 4 Transition effects

And, of course, you can change the way slides move.

So far, these are all the transition effects we've implemented:

  • slide (default)
  • fade
  • none
$("#slider4").responSlider({
    effect: "fade",
    previousSlideAction: "#previousSlide4",    
    nextSlideAction: "#nextSlide4",
    horizontallyCentered: true,
    verticallyCentered: true
});

Example 5 Text slider

You can place whatever you want inside the slider. For example, let's make a Lorem-impsum slider! 

<div id="slider5" class="mySlider">

  <p>Lorem ipsum...</p>

  <p>Eos vocibus...</p>

  ...

</div>
$("#slider5").responSlider({
    visibleSlides: 3,
    previousSlideAction: "#previousSlide5",    
    nextSlideAction: "#nextSlide5"
});
